A fully responsive, multi-page coffee shop website I designed and built entirely from scratch. My goal was to push beyond basic static layouts and challenge myself to replicate the kind of polished, interactive experience you'd expect from a real-world business website, using only the core web technologies.

---
## 🚀 Features

🏠 Multi-page layout with a dedicated order page </br>
📌 Sticky navbar with frosted glass blur effect and smooth scrolling </br>
🎬 CSS zoom-out animation on page load </br>
🟢 Live shop open/closed status based on user's local time

---

## 🛠 Built With

- HTML5  
- CSS3 (Flexbox, Grid, transitions, animations)  
- Vanilla JavaScript (ES6+)  
- Intersection Observer API  
- Fetch API  
- Open-Meteo API (weather integration)  
- Formspree (order form handling)

---

## 💡 Key Features

### 🟢 Live Shop Status
Automatically detects whether the café is open or closed based on local time and updates the interface dynamically.

### 🌦 Weather Mood Integration
Fetches real-time weather data and adjusts the site’s mood message to match the current conditions.

### 🎞 Scroll Animations
Uses Intersection Observer to smoothly reveal menu items as they enter the viewport.

### ⭐ Testimonial Carousel
Custom-built slider with arrow navigation and dynamic indicator dots.

### 📦 Online Order Page
Multi-selection form allowing users to choose drinks and artisanal bites.
