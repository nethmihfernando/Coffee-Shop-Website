# Café Bella | Where Beauty Meets Brew

A modern, elegant artisan coffee shop website featuring a dynamic menu, real-time shop status, and a custom delivery ordering system.

## ✨ Features
* **Dynamic Menu**: Categorized brews and bites with interactive hover effects.
* **Real-time Status**: Displays if the shop is open or closed based on local time.
* **Custom Ordering**: A dedicated ordering page with multiple-choice selections for delivery.
* **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.

## 🛠️ Built With
* HTML5
* CSS3 (Custom properties and Flexbox/Grid)
* JavaScript (ES6+)

## 🚀 Local Setup
1. Clone the repository.
2. Open `index.html` in your browser.
3. To test the order form, replace `your_id` in `order.html` with a valid Formspree ID.
