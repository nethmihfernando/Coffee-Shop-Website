A fully responsive, multi-page coffee shop website built using **HTML, CSS, and Vanilla JavaScript**, designed to simulate a refined artisan café experience with real-world interactive features.

---

## ✨ About The Project

Café Serafina is inspired by romantic European cafés — floral notes, artisan pastries, and soft luxury aesthetics.

The website includes:

- A welcoming hero section with brand storytelling  
- A fully designed menu with handcrafted items  
- A live shop status indicator  
- Weather-based mood updates using an API  
- Scroll animations for menu items  
- A testimonial slider with navigation controls  
- A separate online order page  

The intention behind this project was to move beyond beginner HTML/CSS layouts and explore interactive UI behavior using vanilla JavaScript.

---

## 🛠 Built With

- HTML5  
- CSS3 (Flexbox, Grid, transitions, animations)  
- Vanilla JavaScript (ES6+)  
- Intersection Observer API  
- Fetch API  
- Open-Meteo API (weather integration)  
- Formspree (order form handling)

---

## 💡 Key Features

### 🟢 Live Shop Status
Automatically detects whether the café is open or closed based on local time and updates the interface dynamically.

### 🌦 Weather Mood Integration
Fetches real-time weather data and adjusts the site’s mood message to match the current conditions.

### 🎞 Scroll Animations
Uses Intersection Observer to smoothly reveal menu items as they enter the viewport.

### ⭐ Testimonial Carousel
Custom-built slider with arrow navigation and dynamic indicator dots.

### 📦 Online Order Page
Multi-selection form allowing users to choose drinks and artisanal bites.
